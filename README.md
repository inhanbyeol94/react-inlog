# Default React Package
아래의 라이브러리 패키지를 포함하여 구성된 프로젝트입니다.

### Library
- React Query
- Tailwind CSS
- React Router Dom
- Zustand
- Axios
- React Icons

### install
```bash
yarn add @tanstack/react-query
yarn add tailwindcss
yarn add react-router-dom@6
yarn add zustand
yarn add axios
yarn add react-icons
```
